# Level 1: Back to Basics

**Theme:** Scrapyard
**Background:** Chalkboard (Solid dark green)
**Terrain:** Marble tiles (Floor uses the Marble block rather than stone)

## Starting Conditions
- **Currency:** 4 Byts
- **Inventory:** 1x Hand Drill, 10x Stone
- **Grid Setup:** 5 Lanes, Default Ores, Default Difficulty

## Gameplay Flow
1. **Initial Shop:** Upon starting, the player will immediately load into a shop prompt.
2. **Purchasing Mods:** Players can exchange Byts (currency) for Mod Chips.
3. **Core Mods:** The very first shop guarantees three SD cards (Core Mods) that will permanently modify the run. 
4. **Subsequent Shops:** After the initial phase, shops will prompt after each wave and sell any unlocked mods (up to 3) from the global pool.

## Wave Structure
- **Total Waves:** 7
- **Day Phases (Minutes of planning before each wave):**
  - Wave 1: 2 minutes (120s)
  - Wave 2: 4 minutes (240s)
  - Wave 3: 2 minutes (120s)
  - Wave 4: 2 minutes (120s)
  - Wave 5: 2 minutes (120s)
  - Wave 6: 4 minutes (240s)
  - Wave 7: 4 minutes (240s)
